## REQUIREMENTS TO INSTALL
### Install Ollama
In this project we run Ollama's model locally, below is the command to install it in Linux OS.
```bash
curl -fsSL https://ollama.com/install.sh | sh
```
### Python Dependencies
In order to run this project, the following python dependencies need to be installed in the environment:
```bash
pip install chromadb
pip install watchdog
pip install mcp
```
